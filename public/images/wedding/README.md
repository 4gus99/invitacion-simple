To replace with real photos:
1. Put your files in `public/images/wedding/`.
2. Name them exactly: `table.jpg`, `landscape.jpg`, `moment.jpg`, `bouquet.jpg` (or keep the .png extension).
3. The site will automatically load `/images/wedding/<name>` when you refresh the dev server.

Tips:
- Use 4:5 aspect ratio (e.g., 2400x3000) for best visual fit.
- Export as high-quality JPEG (sRGB) and keep the same filenames.
